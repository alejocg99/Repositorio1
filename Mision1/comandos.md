# Comandos
> `git --version` //version del git instalada desde terminal <br>
> `code .` // ejecutar visual code desde CMD <br>
> `git init` // iniciar un repositorio vacio en git <br>
> `git status` //para verificar el status de git<br>
> `git add .` //adicionar archivos al repositorio git <br>
> `git config --global user.email "click.bienesraices7@gmail.com"` // para enlazar el git con el correo del github <br>
> `git config --global user.name "alejocg99"` //dar el nombre del usuario al git <br> 
> `git commit -m "first commit"` // check point del repositorio, tome los cambios en la configuracion <br>
> `git branch -M main` //definir la rama del proyecto github ( si solo se ejecuta git branch, toma la rama master) <br>
> `git remote add origin https://github.com/clickbienesraices818/BCMartinCastano.git` // ligar el proyecto en GitHUb <br>
> `git push -u origin main (master)` // enviar al repositorio y copnfirmar la propiedad <br>
> `git clone URL` //clonar un repositorio de Git Hub de la URLcd .. <br>
`python --version` para verificar la version de python <br>
